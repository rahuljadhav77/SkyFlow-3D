# Sky Flow 3D 🕊️✨

An immersive, cinematic 3D flight simulation that takes you through the layers of the atmosphere—from the sun-drenched meadows to the silent, starry exosphere.

## ✨ Features
- **GPU-Powered Flight**: Simulates 4,096 sparrows in real-time using Three.js WebGPU compute shaders.
- **Dynamic Atmosphere**: Seamless scroll-triggered transitions through Blue Skies, Cloudy Mists, Sunset Oranges, and Midnight Indigo.
- **Cloud & Star Systems**: 3D sphere-cluster clouds and a 4,000-point starfield that react to the sky's illumination.
- **Micro-Animations**: Real-time wing-flapping physics and responsive "flocking" behavior.
- **Premium Design**: Industry-standard glassmorphic footer and poetic storytelling.

## 🚀 Quick Start

To run this project locally, ensure you have [Node.js](https://nodejs.org/) installed, then follow these steps:

1. **Download the code** and open the terminal in the project folder.
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Start the flight**:
   ```bash
   npm run dev
   ```
4. Open your browser to the URL shown in the terminal (usually `http://localhost:5173`).
